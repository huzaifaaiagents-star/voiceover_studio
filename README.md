# 🎙️ Voiceover Studio — Open Source Script to Audio

Turn any script into a high-quality voiceover in seconds.
Built for YouTubers, video creators, and anyone who needs
audio without paying for expensive tools.

---

## ⚠️ Current Version
This version is built on **OpenAI TTS** — purely for 
testing and demonstration purposes. It's the quickest 
way to get started and see the tool in action.

---

## 🚀 Live Demo
Try it here (no install needed):
👉 https://huzaifaaiagents-star.github.io/voiceover_studio/tts-tool.html
---

## What it does right now
- Paste script → get audio instantly
- 6 different voices to choose from
- Speed control
- Download as MP3
- Runs fully in the browser

## How to use
1. Open the live demo link above
2. Enter your OpenAI API key
3. Paste your script
4. Pick a voice and speed
5. Hit Generate → Download MP3
